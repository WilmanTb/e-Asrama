# e-Asrama
A Native Application Build With Android Studio Java

![Screenshot 2023-10-20 121250](https://github.com/WilmanTb/e-Asrama/assets/148366527/d46f64c2-8aeb-4b1f-9bb9-c342c47ec0e2)

# Project Objective
dormitory application used to manage dormitory residents. The admin will handle all dormitory administration using this application so as to facilitate all administrative matters.

# Libraries
1. Firebase
2. Glider
3. Picasso
4. Zoomage
5. Apache Poi
6. StAX
7. Apache XMLBeans
8. Aalto XML
9. Sendmail
10. CircleImageView

# Project Info
1. UI Rendering : Android UI Kit
2. Android Studio Version : Electric Eel | 2022.1.1 Patch 2
3. Android Studio Theme : Material Theme Github Dark (Dark Ofcourse xD)
4. Min SDK Version : 22
5. Image Type : System Image, PNG, Internet Image

# Project Features
1. Start Application With Root Humberger Navigation Menu
2. Build Constraint in Android Studio XML
3. Build Image Upload
4. Zoomable Image
5. Relatime Update RecyclerView Item
6. Read, Create, Update, Custom Query in Firebase Database
